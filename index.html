CJ Custom Content

A simple Wordpress plugin to add custom code to wp header, footer, pages & posts.

Download this project as a .zip file Download this project as a tar.gz file

Contributors: shibulijack

Requires at least: Wordpress 3.2

Tested up to: 3.3

Stable tag: 2.0

Description

An easy way to add custom code to your object types such as pages and posts.

The goal of this plugin is to help you include custom content right from the admin panel with ease.

Your custom content can be HTML code, Javascript or plain text. It can be included in various parts of the wordpress page/post:

  1. Header [Good for tracking codes such as Google Analytics]
  2. Footer [Ideal for social networking site's integration code]
  3. Body of every page & post
  4. Top (or) Bottom of every single post

With this plugin installed, it is very easy to insert scripts such as analytics track code, social networking sites integration snippet etc…

Installation

  1. Install through the WordPress admin or upload the plugin folder to your /wp-content/plugins/ directory
  2. Activate the plugin through the ‘Plugins’ menu in WordPress
  3. Go to the CJ Custom Content menu under Settings tab
